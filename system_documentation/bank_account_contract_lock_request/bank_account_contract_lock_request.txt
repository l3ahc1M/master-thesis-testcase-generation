A request to maintain locks on product features and transactions of a bank account contract. A lock request manages locks for features, bank account entries (transactions), direct debit mandates, creditor parties, and checks. A bank account contract lock request is based on an event type which can be maintained in customizing. This event type defines which objects can be locked and some default values.
You can use this API to create, query,read, update, or delete locks.Comments: Bank Account Contract Bank Account Contract is a contract between a credit institute and customer, which is based on the request of a customer for opening a bank account and consists among others, the bank account type, account holder, general terms and conditions. Current Account Contract, Deposit Account Contract, and Loan Contract are types of Bank Account Contract. Remark: A prerequisite of the API is that business partner data is available in same system as the bank account contracts.The API queries data by the current date of a bank account contract. A Bank Account Contract Lock Request can be used for business transactions like death of account holder and checks lost.The service Bank Account Contract Lock Request is assigned to service group Transactional Banking.
