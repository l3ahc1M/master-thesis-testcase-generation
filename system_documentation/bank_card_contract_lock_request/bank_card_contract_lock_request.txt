A request to maintain locks on product features and transactions of a bank card contract. A Bank Card Contract is based on an event type which can be maintained in customizing. This event type defines which objects can be locked and some default values. You can use this API to create, query, read or update locks.Note:
A Bank Card Contract is a contract between a credit institute and customer, which is based on the request of a customer to have a bank card for their bank account and consists among others, the bank card type, card holder, general terms and conditions. A Bank Card Contract Lock Request can be used for business events like locking renewal of the card. A prerequisite of the API is that business partner data is available in same system as the bank card contracts.Remark:
The service Bank Card Contract Lock Request is assigned to service group Transactional Banking.
